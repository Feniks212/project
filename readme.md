### Installation

BeSnowSure requires [Node.js](https://nodejs.org/) v4+ to run.

Install the dependencies and devDependencies and start the server.

```sh
$ cd project_BeSnowSure
$ npm install -g gulp
$ npm install -d
```

Nex step...

```sh
$ gulp
```
